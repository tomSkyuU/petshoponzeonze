# 🐾 Pet Shop OnzeOnze — Site Institucional

Site completo para o Pet Shop OnzeOnze, localizado em Pelotas – RS.

## ✨ Funcionalidades

- Catálogo com 30+ produtos reais (Royal Canin, Hill's, Frontline, Kong, Purina…)
- Carrinho de compras com controle de quantidade
- Opção de entrega ou retirada na loja
- Cálculo automático de frete (grátis acima de R$150)
- Finalização de pedido via WhatsApp
- Filtros por categoria (cães, gatos, aves, peixes, roedores)
- Seção de serviços com preços (banho & tosa, veterinário, vacinas…)
- Avaliações de clientes
- Layout responsivo (mobile e desktop)

## 🛠️ Tecnologias

- HTML5, CSS3, JavaScript puro (sem frameworks)
- Google Fonts (Bebas Neue + Nunito)
- Deploy: Vercel

## 🚀 Como rodar

Abra o `index.html` direto no navegador — sem instalação.
